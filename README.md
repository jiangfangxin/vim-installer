# auto-config-vim

## 適合對象

- 常常需要在新的環境coding，卻懶得重新配置vim

- 怕自己亂搞把環境搞壞(?

- 單純想用airline卻不會安裝的同胞

## 介紹

![](https://i.imgur.com/GdzjHUv.gif)

- 支援系統
  - ubuntu

- 自動安裝 
  - git
  - vim
  - vundle (vim套件管理工具) [github](https://github.com/VundleVim/Vundle.vim)
  - airline (vim環境中高亮度顯示模式) [github](https://github.com/vim-airline/vim-airline)

## 使用說明

把檔案下載到目錄

在目錄中執行install.sh
> ./install.sh
